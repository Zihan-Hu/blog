### 找众数战士

给定一个长度为 $n$ 的序列，你需要对于每个前缀，找到出现次数最多的那个数字。如果有多个出现次数最多的数字，输出最小的那个。

由于输出的数字太多了，为了加快输出，你只需要输出所有数字的**异或和**。

### 输入格式

第一行一个正整数 $n$。

第二行 $n$ 个正整数 $a_i$，表示序列。

**由于输入量较大，建议使用快速输入。**

### 输出格式

一个非负整数，表示输出数字的异或和。

### 样例

输入：

```
7
1 3 2 2 1 2 1
```

输出：

```
1
```

### 数据范围

对于 100% 的数据，$1 \leq a_i \leq n$。

- 测试点 1~3：$n \leq 10^3$
- 测试点 4~7：$n \leq 10^5$
- 测试点 8~10：$n \leq 10^7$
